\# TRABAJOFINAL



Proyecto TFG desarrollado con PHP y MySQL (XAMPP).



\## Requisitos

\- XAMPP

\- PHP 8.x

\- MySQL / phpMyAdmin



\## Instalación

1\. Clona el repositorio dentro de `htdocs`

2\. Importa la base de datos desde `sql/init.sql`

3\. Copia `app/config.example.php` a `app/config.php`

4\. Ajusta las credenciales de la base de datos

5\. Accede desde el navegador a:

&nbsp;  http://localhost/TRABAJOFINAL/public/index.php
